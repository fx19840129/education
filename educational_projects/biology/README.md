# 生物学科项目

## 项目概述
生物学科项目专注于物种分类、生态系统模拟、细胞结构可视化等方面的教育内容。

## 包含项目
- **species_classifier** - 物种分类器
- **ecosystem_simulator** - 生态系统模拟器
- **cell_structure_visualizer** - 细胞结构可视化工具
- **genetics_explorer** - 遗传学探索器
- **evolution_tracker** - 进化追踪器

## 技术特点
- 基于核心工具模块的统一架构
- 专注于生物内容生成和模拟
- 支持多种生物学习场景
- 可配置的模拟参数和难度

## 使用方法
每个项目都可以独立运行，生成相应的生物学习资源。
