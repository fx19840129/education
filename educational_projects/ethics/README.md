# 道德法制学科项目

## 项目概述
道德法制学科项目专注于案例分析、规则解释、法律条文等方面的教育内容。

## 包含项目
- **case_analyzer** - 案例分析器
- **rule_explainer** - 规则解释器
- **law_article_generator** - 法律条文生成器
- **moral_dilemma_simulator** - 道德困境模拟器
- **civic_education_builder** - 公民教育构建器

## 技术特点
- 基于核心工具模块的统一架构
- 专注于道德法制内容生成和分析
- 支持多种道德法制学习场景
- 可配置的案例类型和难度

## 使用方法
每个项目都可以独立运行，生成相应的道德法制学习资源。
